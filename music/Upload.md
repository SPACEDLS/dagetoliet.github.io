# 音樂上傳
fork該倉庫到您的賬戶中，修改克隆倉庫，上傳音樂文件至該目錄下/src/，將封面圖片上傳至/img/mus/，然後在mus.json裡面添加音樂信息：
```json
 {
    "title": "音樂名稱 例如 大哥東",
    "artist": "作曲者 例如 大哥",
    "cover": "封面地址 例如 (./img/mus/mus1.png)",
    "url": "音樂地址，本地目錄即可 例如（./src/s.mp3)"
  }
```
上傳到你的克隆倉庫後，申請一下 PullRequests，等待審核完成。
## 提醒
該站點僅支持上傳有關“大哥”音效的內容，其他音頻恕無法上傳。
## 侵權處理
如果本站音頻侵權，請聯絡倉庫管理者刪除音頻。
### 大哥國際廁所集團  本站僅供娛樂請勿當真內容虛構 音頻資源由用戶上傳，如侵權請聯絡刪除。
